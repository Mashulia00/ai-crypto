# ML-сигналы для крипторынка: обзор

Модель классифицирует рыночные **режимы** (тренд / флэт / высокая волатильность) и возвращает вероятности.  
Решения по сделкам принимаются по правилам, а не под влиянием эмоций.

## Как мы избегаем переобучения
- Честное разделение данных и кросс-валидация
- Жёсткие ограничения: стоп-лоссы, тейк-профиты, риск на сделку
- Мониторинг качества в реальном времени (демо)

> Прошлые результаты не гарантируют будущую доходность
